## Add capabilities on demand via proven plugins (MCP)
- MCP are "plugins" for AI tools ‚Äî connect to services in a click.

- üîé Browse MCP servers: https://mcp.so/ ‚Äî pick one to add via Codex CLI.

- Comes with: [Firecrawl](https://github.com/firecrawl/firecrawl) and [Context7](https://github.com/upstash/context7)

- Prompts:
  - `"Find an MCP for <your need>"`
  - `"Will an MCP help us <your goal>?"`
  - `"Install the <X> MCP server"`

## You ask for outcomes; Codex installs what‚Äôs needed behind the scenes
- üß† Smart installs ‚Äî When you ask for something new (e.g., ‚Äúexport to Excel‚Äù, ‚Äúrender a PDF‚Äù, ‚Äúsave as Parquet‚Äù), Codex CLI installs the right Python packages automatically using `uv`.

- üîí Safe & local ‚Äî Everything lives in the project‚Äôs virtual environment; no global `pip`. Dependencies are tracked in `pyproject.toml` + `uv.lock` for repeatability.

- üßπ Clean by default ‚Äî After adding deps, Codex runs project checks to keep things tidy.

- Examples:
  - ‚ÄúRender a PDF report‚Äù ‚Üí adds `weasyprint` (or `pdfkit`) and configures the export.
  - ‚ÄúExport results to Excel‚Äù ‚Üí adds `openpyxl` (or `xlsxwriter`).
  - ‚ÄúSave as data tables for querying‚Äù ‚Üí adds `pyarrow` for fast columnar files.
  - ‚ÄúPlot a quick chart‚Äù ‚Üí adds `plotly` (or `matplotlib`) when needed.
  - ‚ÄúFetch and parse a page‚Äù ‚Üí adds `httpx` + `beautifulsoup4` for lightweight scraping.

- You just ask; Codex brings the pieces together so you can focus on outcomes.

## Get Started Now

> First step for windows users only
- Windows (first‚Äëtime)
  - Download the `windows-setup.ps1` script from [**here**](scripts/windows-setup.ps1)
  - Open PowerShell as Administrator
  - Run: `./scripts/windows-setup.ps1 -ProvisionWSL -DefaultProfile Ubuntu`
  - Reboot if prompted, open Ubuntu (WSL), and work inside your Linux home (e.g., `~/`)

- Get the code
  - Clone: `git clone https://github.com/Sharper-Flow/super-codex-workbench.git`
  - Or fork on GitHub, then: `git clone https://github.com/<your‚Äëusername>/super-codex-workbench.git`
  - Enter the folder: `cd super-codex-workbench`

- Install Codex CLI
  - Ensure ChatGPT Plus/Pro access
  - Follow the official Codex CLI install guide for your OS
  - Verify it launches and can open this repo workspace

- Launch Codex CLI and Run Setup
  - Open your terminal (or WSL2 Ubuntu on Windows)
  - Run: `codex`
  - Once Codex CLI is running, tell it:
    > "run the setup script"
